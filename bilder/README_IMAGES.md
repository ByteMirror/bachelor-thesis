# Benötigte Bilder für Kapitel 2.1

## VTT-Komponenten Visualisierungen

### 1. vtt-token-example.png
- **Beschreibung**: Split-Screen Darstellung von Token
- **Links**: Spieleransicht mit Token (nur Name und HP-Balken sichtbar)
- **Rechts**: GM-Ansicht mit vollständigen Stats
- **Empfohlene Quelle**: Screenshot aus Roll20 oder Foundry VTT

### 2. statblock-daggerheart.png
- **Beschreibung**: Beispiel-Statblock aus Daggerheart System
- **Inhalt**:
  - Oben: Name und Beschreibung der Kreatur
  - Mitte: Attribute (Strength, Agility, etc.)
  - Unten: Verfügbare Aktionen und Fähigkeiten
- **Alternative**: D&D 5e Statblock oder Pathfinder 2e

### 3. fog-of-war-comparison.png
- **Beschreibung**: Vergleich GM vs Spieler Sicht
- **Oben**: GM-Ansicht mit vollständiger Karte
- **Unten**: Spieleransicht mit:
  - Schwarze Bereiche (unerforschte)
  - Graue Bereiche (erforschte aber nicht sichtbar)
  - Helle Bereiche (aktuell sichtbar)

### 4. initiative-tracker.png
- **Beschreibung**: Initiative-Tracker Interface
- **Inhalt**:
  - Sortierte Liste der Kampfteilnehmer
  - Hervorhebung des aktuell Handelnden
  - HP-Balken und Status-Icons
  - Runden-Counter

### 5. dice-roller-3d.png
- **Beschreibung**: 3D-Würfel Animation
- **Inhalt**:
  - Mehrere polyedrische Würfel mid-roll
  - Ergebnis-Anzeige
  - Formel-Eingabefeld (z.B. "3d6+2")
